# Setup Guide

## Installation & Setup

=== "Windows"

    **1. Install Python**
    1. Download the installer from [python.org](https://www.python.org/downloads/).
    2. Run it.
    3. **Important:** Check **"Add python.exe to PATH"** at the bottom.

    **2. Verify**
    Open `cmd` and run:
    ```cmd
    python --version
    ```

=== "macOS"

    **1. Install Python**
    1. Download the **macOS 64-bit universal2 installer**.
    2. Run the package installer.

    **2. Verify**
    Open `Terminal` and run:
    ```bash
    python3 --version
    ```

=== "Linux"

    **1. Install Python**
    ```bash
    sudo apt update
    sudo apt install python3 python3-pip python3-venv
    ```

    **2. Verify**
    ```bash
    python3 --version
    ```
